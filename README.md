# 🛡️ أكاديمية الكيلاني للتداول الذكي | El-Kilany Trading Academy

منصة تعليمية وتدريبية متكاملة ومتقدمة لتعليم التداول والتحليل الفني في الأسواق المالية، مدمجة بمدرب ذكاء اصطناعي ذكي ومحاكي تداول تفاعلي متطور لمحاكاة الأسواق الحقيقية.

---

## 🚀 المميزات الرئيسية للمنصة

تحتوي الأكاديمية على ستة أقسام برمجية وتفاعلية متكاملة لتقديم تجربة تعليمية فائقة الجودة:

### 1. 🏫 الفصول والمسارات التعليمية (Lecture Classroom)
* تنظيم الدروس والمحاضرات الفنية في مسارات منظمة ومستويات (أساسي، متقدم، احترافي).
* إمكانية إرفاق ملفات PDF وملفات تدريبية لكل محاضرة مع ميزة التنزيل الفوري.
* نظام لتقديم طلبات الاشتراك وإثباتات الدفع ورفع صور إيصالات التحويل لمراجعتها من قبل الإدارة.

### 2. 📈 محاكي التداول الذاتي التفاعلي (Trading Simulator)
* تداول حقيقي يحاكي واقع السوق ومخططات الشموع الحية باستخدام **رصيد افتراضي بقيمة 10,000$**.
* تنفيذ فوري لصفقات الشراء والبيع والتحكم في حجم العقد (Lot size)، مع حساب تلقائي للأرباح والخسائر والـ Drawdown.
* ميزة **نسخ صفقات المشرف الفني (Copy Trading)** وربط حساب المتداول بسيرفر نسخ الصفقات لأستاذ الكيلاني بنجاح.

### 3. 🤖 مستشار ومخطط الذكاء الاصطناعي (AI Coach)
* مدعوم بموديل **Gemini AI** المتطور لتحليل الصفقات وتقديم نصائح واستشارات مالية حية واستراتيجيات تداول حصرية.
* دفتر ملاحظات رقمي ذكي لتدوين صفقاتك واستراتيجياتك والاحتفاظ بنصائح الذكاء الاصطناعي مدمج مباشرة بالمنصة.

### 4. 📒 قاموس الشموع اليابانية التفاعلي (Candlesticks Dictionary)
* مرجع شامل لجميع نماذج الشموع اليابانية الفنية (المطرقة، شمعة الابتلاع، دوجي، وغيرها) مع شروحات مبسطة للتنبؤ بحركة السعر بدقة.

### 5. 🛡️ لوحة التحكم والإدارة الفنية للأستاذ فتحي الكيلاني (Admin Panel)
* لوحة تحكم سرية ومطورة مخصصة للأستاذ فتحي الكيلاني يتم الدخول إليها برمز مرور مخصص أو عبر **البصمة الحيوية الفنية الذكية**.
* مراجعة طلبات الانضمام والاشتراكات للأعضاء الجدد، وتعديل حالة الطلب (قبول/مسودة).
* إمكانية حظر وطرد الأجهزة والوصول غير المصرح به بنقرة زر واحدة لحماية محتوى الأكاديمية.
* تعديل وإدارة طرق الدفع، ورفع المحاضرات والمسارات الجديدة وتتبع الأرباح وسحبها الفوري.

### 6. 📱 دعم التطبيقات التقدمية (PWA Install)
* التطبيق مهيأ ومزود بـ **Service Worker** لدعم التنزيل وتثبيت المنصة مباشرة على هواتف الأندرويد، الآيفون، أو أجهزة الحاسوب الشخصية والوصول إليها كتطبيق محلي كلياً.

## 🛠️ البنية البرمجية والتقنيات المستخدمة (Tech Stack)

### الواجهة الأمامية (Frontend):
* **React 19 & TypeScript 5+**: لبناء واجهة تفاعلية ديناميكية خالية من الأخطاء النوعية تماماً.
* **Vite & @tailwindcss/vite**: لتوفير جيل جديد من التصميم الفائق والمرن باستخدام ومكونات فلويد تفاعلية (Cairo & Inter font).
* **Lucide React & Motion**: أيقونات تداول فنية راقية وحركات خفيفة ومحاذاة ممتازة لخيارات القوائم.
* **Recharts**: لرسم وتحليل مخططات التداول والاتجاهات البيانية ومسارات الأرباح التراكمية.

### الخلفية والخدمات (Backend):
* **Node.js & Express 4**: لتوفير خادم محلي كامل لإدارة الجلسات الحية والتحقق من رموز المسؤول ومصادقة البصمة الحيوية.
* **Gemini @google/genai SDK**: الاتصال بأقوى النماذج الذكية لتوفير مدرب ومحلل ذكي يدعم العربية بطلاقة.

## 🚀 تشغيل وتطوير المشروع محلياً

اتبع الخطوات التالية لتشغيل الأكاديمية على جهازك المحلي:

### 1. تثبيت الحزم والمكونات:
```bash
npm install
```

### 2. إعداد متغيرات البيئة:
قم بإنشاء ملف `.env` في المسار الرئيسي وتكوينه كالتالي:
```env
GEMINI_API_KEY="أدخل_مفتاح_جيميني_الخاص_بك"
APP_URL="http://localhost:3000"
```

### 3. تشغيل خادم التطوير المباشر:
```bash
npm run dev
```
سيتم بدء تشغيل المنصة على عنوان المنفذ المحلي: http://localhost:3000

### 4. بناء نسخة الإنتاج الجاهزة للنشر:
```bash
npm run build
npm run start
```
